Installation Guide

Follow these steps to start your adventure:

Requirements:

  • Python 3.x installed on your computer
  • Colorama package (for colored text output)
  1. Install the required package by opening your terminal/command prompt and running: pip install colorama
  2. Navigate to the directory where you saved the game file
  3. Run the game using: python game.py
Note: If you encounter any issues running the game, make sure Python is properly installed and added to your system's PATH.

Game Controls

  • Use numbers (1, 2, 3) to make your choices
  • Press Enter to confirm your selection
  • Type 'quit' at any time to exit the game
